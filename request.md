# Request BMI Analyzer

## Tujuan
Saya ingin membuat aplikasi web untuk menghitung Body Mass Index (BMI) dan memberikan analisis kesehatan dasar yang mudah dipahami pengguna.

## Stack Wajib
- Next.js (JavaScript / vanilla JS)
- Tailwind CSS
- shadcn/ui

## Fitur Fase Awal
### Input pengguna
Aplikasi perlu menerima data yang dibutuhkan untuk menghitung BMI dan analisis nutrisi, minimal:
- tinggi badan
- berat badan
- usia
- jenis kelamin
- tingkat aktivitas

### Output dan analisis
Setelah pengguna mengisi data, aplikasi harus dapat:
- menghitung BMI
- menampilkan kategori BMI
- menghitung berat badan ideal
- menghitung kebutuhan kalori harian
- menghitung kebutuhan protein harian
- menghitung kebutuhan lemak harian
- menampilkan penjelasan singkat mengenai hasil BMI
- menampilkan tips umum untuk membantu pengguna mencapai berat badan ideal

### Fitur akun
Pengguna juga dapat:
- membuat akun
- login
- menyimpan hasil perhitungan
- melihat riwayat hasil perhitungan

### Aturan akses
- pengguna dapat menghitung dan melihat hasil tanpa login
- pengguna wajib login hanya jika ingin menyimpan hasil perhitungan
- pengguna wajib login hanya jika ingin mengekspor hasil analisis

### Fitur export
Pengguna dapat mengekspor hasil analisis dalam format:
- JPG
- PDF

## Fitur Fase Lanjutan
Jika fase awal sudah stabil, aplikasi bisa dikembangkan dengan fitur tambahan:
- rekomendasi menu diet rumahan
- rekomendasi yang disesuaikan dengan domisili atau wilayah pengguna
